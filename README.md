# Japanese Learning Center

## How to use

### Prerequirement
Install Node and Npm

### How to install
`git clone https://github.com/hiroTochigi/japanese`
`cd japanese`

#### Install Front End
1. Go to front directory
1. `npm install`
1. `npm start`

#### Install proxy server
1. Go to back directory
1. `npm install`
1. `npm start`

Then go to http://localhost:3000/
